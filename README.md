# pokeAgenda
Primeiramente crie um ambiente .venv seguindo os passos:
Ctrl+Shift+P --> Python: create environment --> .venv
Depois certifique-se se o terminal está com o .venv selecionado
Depois dê um pip install -r requirements.txt
Depois inicie o servidor XAMPP e crie um db com o nome "pokebanco" e cole o sql do arquivo pokebanco.sql
Depois dê um python app.py no terminal e aproveite!
